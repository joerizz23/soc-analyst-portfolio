# SOC Alert Triage Lab

## Overview
This project demonstrates my ability to investigate and triage common SOC alerts using a structured incident analysis workflow.

The lab focuses on:
- validating alerts
- analyzing suspicious activity
- determining severity
- recommending containment and response actions

## Objectives
- Practice analyst-style alert triage
- Document findings clearly
- Show escalation thinking
- Build repeatable incident investigation habits

## Scenarios Included
1. Brute-force login attempt
2. Suspicious PowerShell execution
3. Impossible travel login

## Triage Workflow
1. Review alert details
2. Validate whether activity is suspicious
3. Gather supporting evidence
4. Determine impact and severity
5. Recommend next steps

## Skills Demonstrated
- SOC alert triage
- Incident investigation
- Severity classification
- Threat reasoning
- Documentation and reporting

## Tools
- Microsoft Sentinel / Splunk
- Windows event logs
- KQL or SPL
- VirusTotal
- CyberChef
